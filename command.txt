# 해당 명령어는 관리자 기준

#----------------------------------------------------------------------------------#

# (src 폴더에서)

1. 커널 소스 빌드

make

2. module 등록 및 생성
   (여기서 MAJOR_NUMBER(== 100) 가 다른 driver 와 겹치지 않는지 꼭 확인 할 것)

> insmod my_device.ko
> mknod /dev/my_device c 100 0

#----------------------------------------------------------------------------------#

# (run 폴더에서)

3. application 빌드
  (여기서 arm 보드 기준이므로, arm 용 컴파일러를 이용해 빌드해야 함)

> arm-linux-gnueabihf-gcc run.c
